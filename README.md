A demonstration mobile application developed in Android Studio using Java. This project integrates the Gemini API to provide chatbot functionality.

* Supports text, audio, and image input
* Maintains chat history


1. Fork the repository and open it in Android Studio.
2. Create a free Gemini API key from **Google AI Studio**.
3. Open `gradle.properties` and paste your API key:

   ```
   GEMINI_API_KEY=your_api_key_here
   ```
4. Build and run the application.
